# Overview

Программа «Блокнот»: ввод 10 записей типа NOTE (фамилия и инициалы, телефон, дата рождения), упорядочивание по возрастанию даты рождения, вывод списка и поиск по номеру телефона. Реализована по модели КИС-2 (клиент — интерфейс — сервер). Исходники в `src/`, заголовки в `include/`, сборка через `make`.

# Usage

Сборка: `make`. Запуск: `./blocknote` или `make run`. Программа запрашивает 10 записей, выводит отсортированный список и номер телефона для поиска. Для неинтерактивного теста: `./blocknote < tests/test_input.txt`.

# Additional Notes

CI (GitHub Actions) при push и pull_request выполняет `make` и запуск с тестовым вводом. Контрольные расчёты и тесты — в `docs/test_report.md`.
