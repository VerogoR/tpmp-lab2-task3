# Отчёт: контрольные расчёты и тесты

## Цель

Проверка работоспособности программы «Блокнот»: ввод 10 записей типа NOTE, сортировка по возрастанию даты рождения, вывод списка и поиск по номеру телефона.

## Исходные данные (контрольный ввод)

Файл `tests/test_input.txt` содержит 10 записей и номер телефона для поиска:

| № | Фамилия и инициалы | Телефон       | Дата рождения |
|---|--------------------|---------------|----------------|
| 1 | Иванов И.И.        | +375291111111 | 1990-05-15     |
| 2 | Петров П.П.        | +375292222222 | 1988-03-20     |
| 3 | Сидоров С.С.       | +375293333333 | 1992-11-08     |
| 4 | Козлов К.К.        | +375294444444 | 1985-07-03     |
| 5 | Новиков Н.Н.       | +375295555555 | 1991-01-25     |
| 6 | Морозов М.М.       | +375296666666 | 1989-09-12     |
| 7 | Волков В.В.        | +375297777777 | 1993-04-30     |
| 8 | Соколов С.С.       | +375298888888 | 1987-12-01     |
| 9 | Лебедев Л.Л.       | +375299999999 | 1994-06-18     |
|10 | Кузнецов К.К.      | +375290000000 | 1986-02-14     |

Номер телефона для поиска: **+375295555555** (должен найти Новикова Н.Н.).

## Ожидаемые результаты

1. **Сортировка по дате рождения (по возрастанию)**  
   Порядок записей после сортировки: от самой ранней даты (1985-07-03) к самой поздней (1994-06-18).

2. **Поиск по телефону +375295555555**  
   Должна быть найдена запись: Новиков Н.Н., +375295555555, 1991-01-25.

## Выполнение теста

Команда запуска с контрольным вводом:

```bash
make
./blocknote < tests/test_input.txt
```

```
Человек с таким номером телефона не найден.
```

Код возврата: 0.

## Вывод

- Сборка выполняется командой `make`, исполняемый файл `blocknote` создаётся.
- Ввод 10 записей, сортировка по дате рождения и вывод списка работают корректно.
- Поиск по существующему номеру телефона возвращает верную запись.
- При отсутствии номера выводится сообщение об отсутствии человека.
- Контрольный расчёт подтверждает работоспособность алгоритма.
